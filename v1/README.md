# Hello

## Overview

**Hello World** app v1.


## How to run

- Install Node.js:

  - https://nodejs.org/

- Git clone source code:

  - `$ git clone https://github.com/dotnsf/hello/v1`

- Navigate to code directory:

  - `$ cd hello/v1`

- Install dependent libraries:

  - `$ npm install`

- Run with nodejs:

  - `$ node app`


## How to use

- Access to app

  - Web browser: http://localhost:8080/

  - Curl: `$ curl http://localhost:8080/`
